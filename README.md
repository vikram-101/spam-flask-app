 # 🛡️ SMS Spam Detector

A Machine Learning web application that detects whether an SMS message is **Spam** or **Not Spam** using Natural Language Processing (NLP).

## 🚀 Live Demo
👉 [Click here to view the app](https://spam-flask-app-3.onrender.com)

## 🛠️ Tech Stack
- **Python** — Core language
- **Flask** — Web framework
- **Scikit-learn** — ML model (Naive Bayes)
- **TF-IDF Vectorizer** — Text preprocessing
- **HTML & CSS** — Frontend
- **Render** — Deployment

## 📁 Project Structure
```
spam-flask-app/
├── app.py
├── spam.py
├── templates/
│   ├── home.html
│   └── result.html
├── static/
│   └── style.css
├── NB_spam_model.pkl
├── vectorizer.pkl
├── requirements.txt
└── Procfile
```

## ⚙️ How It Works
1. User enters an SMS message
2. Message is vectorized using TF-IDF
3. Naive Bayes model predicts Spam or Not Spam
4. Result is displayed instantly
